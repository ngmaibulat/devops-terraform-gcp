# Terraform with GCP

### Prerequisites

-   terraform
-   gcloud
-   GCP project created
-   GCP service account created

### GCP Labs

-   VPC, with subnets in 2 regions
-   VMs in 2 subnets/regions
-   DNS
-   Load Balancer + VM
-   Load Balancer + Bucket
-   TLS
-   CDN
-   Provisioners
-   Metadata
-   FileStore
-   FireStore
-   Persistent Disk
-   Managed Instance Group, AutoScaling
-   Cloud Run
-   GKE
